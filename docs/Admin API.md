**Refresh All**
----
  Refreshes all details for all individuals and teams.

* **URL**

  /admin/refresh_all

* **Method:**

  `GET`
  
*  **URL Params**

  NIL

* **Data Params**

  NIL

* **Success Response:**

  * **Code:** 200 <br />
    **Content:** `{
        "message": "Successfully refreshed all teams' mileage",
        "result": {
            "1": 7
        },
        "success": true
    }`
